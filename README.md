# Comp-tool-kit
Welcome to my first ever public project! As someone who loves to edit on adobe after effects and code I figured why not make something that ties into both. There are scripts that already exist out there but I personally dont feel like paying for them when I can just make my own. Ill be constantly updating the file and trying to implement a ui!
Comp tool kit 
Features
⚓ Anchor

3×3 Anchor Point Grid — click any position to snap the anchor point without moving the layer
Center Layer — center to comp both axes, horizontal only, or vertical only
Reset Transform — reset position, rotation, scale and opacity in one click
Reset Anchor Point — snap anchor back to center of the layer

＋ Create

Layer Types — add Null, Solid (with unique colors each time), or Adjustment layers with custom names
Shapes — add Rectangle, Circle, Rounded Rect, or Line shape layers
Text — add a Text layer or a centered Title layer
Camera — add a Camera or a 3D Orbit Null

⚡ Actions

Smart Precomp — precomp selected layers with a custom name
Precomp + Open — precomp and immediately open the new comp
Trim to Comp — trim selected layers to the work area
Duplicate + Offset — duplicate with a 2-frame time offset
Rename Selected — bulk rename with auto-numbering
Delete Hidden Layers — remove all disabled layers from the comp
Auto Parent to Null — create a null and parent selected layers to it
Control Null — add a named control null
Time Pointer — jump the playhead to any frame number


Installation
Run once (quick)

In After Effects go to File → Scripts → Run Script File...
Select CompToolkit.jsx

Dock it permanently

Copy CompToolkit.jsx to:

Windows: C:\Program Files\Adobe\After Effects [version]\Support Files\Scripts\ScriptUI Panels\
Mac: /Applications/Adobe After Effects [version]/Scripts/ScriptUI Panels/


Restart After Effects
Go to Window → CompToolkit.jsx — dock it anywhere you like
